# FLOQR AI Functions Scaffold

This folder is a deployment scaffold only. The GitHub Pages app does not load it.

Planned live backend responsibilities:

- Scheduled public-source discovery crawl 4-6 times per day.
- Ticketmaster, Eventbrite, approved resale partner, official venue/feed ingestion.
- Gemini/Firebase AI Logic structured JSON classification and rating.
- Media understanding/moderation notes.
- ShoutOut copy improvement.
- Template background generation or modification through an approved server-side provider.

Do not place AI API keys in frontend code. Use Firebase Functions, Cloud Run Functions, Secret Manager, and Firebase Auth checks.
