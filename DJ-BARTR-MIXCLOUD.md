# DJ selling on FloqR / BartR + Mixcloud

## Product model (safe defaults)

| Sell on BartR | How it works |
|---|---|
| **Playlist curation** | Track lists + Spotify / Apple Music / Tidal links, BPM/transition notes, themes, schedules. Buyer streams on their own licensed account. **No audio files.** |
| **Mixcloud show** | Mixcloud.com URL + optional Creator Subscriptions link. Card embeds Mixcloud widget. Monetize mixes via Mixcloud’s licensed revenue share. |
| **DJ consultation** | Programming / wedding / gym / restaurant playlist design as a service. |
| **DJ merch** | Physical apparel and swag (normal shipping). |
| **Cleared mix download** | Only with rights cert: own/commissioned, written master+composition clearance, or commercial-resale library. |

## Explicitly blocked / discouraged

- Delivering uncleared recordings (mp3/wav packs) as the product
- Guaranteed playlist placement for artists (Spotify policy)
- Relying on “under 30 seconds”, “promo only”, or “no copyright infringement intended”

## Mixcloud integration (v29.09.12)

1. DJ publishes show on Mixcloud (Pro + Creator Subscriptions for fan monetization).
2. On BartR Store, choose product type **Mixcloud show / mix**, paste show URL (+ optional subscribe URL).
3. Shoppers see Mixcloud embed on BartR; purchase pays FloqR (MoR); DJ fulfills with access notes / merch / consultation as described.
4. FloqR does **not** re-host mix audio. Listening stays on Mixcloud.

Widget docs: https://www.mixcloud.com/developers/widget/  
Creator Subscriptions: Mixcloud Help Center (Pro feature).

## Marketing

Club Admin → Advertising includes DJ Mixcloud / playlist pack SMS/WhatsApp templates. Campaigns debit messaging credits.
