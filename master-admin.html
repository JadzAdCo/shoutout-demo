Feature Diagnostics

Statuses: Pass, Soft Fail, Failed, and TBI. Soft Fail means the feature is present and safe, but has missing optional data, a fallback mode, or a non-blocking setup gap.

Package Install Diagnostics

Firebase Rules Smoke Test

Runs live Firestore, Mingl/chat query, and Storage checks against Firebase. A package rules note can pass while the live deployed Firebase rules still fail if the rules were not published in Firebase Console.

Use this section this way: publish firestore.rules and storage.rules, click Run Rules Smoke Test, then use the suggested fixes or Export Diagnostics TXT if anything fails.

Crawler Search Criteria

Manual runs create reviewable aiDiscoveryQueue records. Real public-source crawling requires backend scheduled functions.

Use one line per market. Each line answers four simple questions: Where? What language? Which cities? What should FLOQR look for?

Format

Country or market | language(s) to search in | cities | event or venue types

Example

Spain | Spanish, Catalan | Barcelona, Madrid, Ibiza | Latin nights, rooftop bars, beach clubs, resale tickets

Example

United Arab Emirates | Arabic, English | Dubai | rooftop lounges, Arabic music DJs, comedy shows, tickets
