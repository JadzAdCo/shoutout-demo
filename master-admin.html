Stale Record Cleanup

Search stale AI discovery queue records and stale club ShoutOut queue records, review why FLOQR considers them stale, and remove them from active review.

A stale record is a queue record older than 4 days, a record that references old Firestore/Storage rules, or a record that points to an old/unknown location. Remove means soft-clear: discovery records get `stale-cache-cleared`; ShoutOut queue records get `stale-shoutout-cleared`. Approved listings, live display content, user profiles, guest lists, and audit history are not changed.

Duplicate Records

Find live Firestore club records that represent the same institution, choose a primary record, and complete/repair the merge. Completion means the duplicate clubLocation points to the primary with canonical fields; the separate alias collection is optional.

Master Admin Settings - Diagnostics

System-wide health, feature status, package checks, and Firebase rules compatibility. AI crawling controls live on the AI Crawling tab.

Feature Diagnostics

Statuses: Pass, Soft Fail, Failed, and TBI. Soft Fail means the feature is present and safe, but has missing optional data, a fallback mode, or a non-blocking setup gap.

Manual Feature Test Diagnostics

Use this checklist after uploading a package. Automated diagnostics prove that files/rules are present; these checks confirm the user-facing feature actually works in the browser. Mark each implemented or changed feature Succeed or Failed, then copy the resolution prompt if anything fails.

Package Install Diagnostics

Checks feature markers added by each package after upload so Master Admin can confirm the installed files match the expected release behavior.

Firebase Rules Smoke Test

Runs live Firestore, Mingl/chat query, and Storage checks against Firebase. A package rules note can pass while the live deployed Firebase rules still fail if the rules were not published in Firebase Console.

Use this section this way: publish firestore.rules and storage.rules, click Run Rules Smoke Test, then use the suggested fixes or Export Diagnostics TXT if anything fails.
